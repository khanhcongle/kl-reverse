== Building

To launch your tests:
```
mvn clean test
```

To package your application:
```
mvn clean package
```

To run your application:
```
mvn clean compile exec:java -Dmaven.test.skip
```

To custom ports:
```
mvn clean compile exec:java -Dmaven.test.skip -Dports.proxy=8080 -Dports.origin=8082
```
